# Program Dashboards

[← Back to REPORTING](../README.md) | [↑ Up to CMP](../../README.md)

Visual performance dashboards for program status.

## Purpose

Provide at-a-glance:
- Health status
- Trend indicators
- Exception highlights
- Action needs

## Dashboard Types

### Executive Dashboard
- Overall health (Red/Yellow/Green)
- Key milestones
- Budget status
- Critical issues

### Technical Dashboard
- Requirements status
- Design progress
- Test results
- Issue resolution

### Manufacturing Dashboard
- Production rate
- Quality metrics
- Supply chain status
- Facilities utilization

## Visualization

- Status indicators
- Trend charts
- Heat maps
- Gantt charts
